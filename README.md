# Tópicos em Desenvolvimento Front-End

Projeto desenvolvido para a disciplina de **Programação Front-End** do curso de
**Análise e Desenvolvimento de Sistemas**.

## Sobre o projeto

Esta aplicação foi desenvolvida em Angular e aborda conceitos importantes do
desenvolvimento front-end moderno.

O projeto possui páginas sobre:

- Angular
- Vue.js
- React
- Componentes Angular
- Consumo de API com HttpClient
- Introdução a testes com Karma e Jasmine

## Tecnologias utilizadas

- Angular
- TypeScript
- HTML
- CSS
- Angular Router
- Angular HttpClient
- RxJS
- Karma
- Jasmine

## Páginas da aplicação

| Página | Rota | Descrição |
|---|---|---|
| Home | `/home` | Apresentação geral do projeto |
| Frameworks | `/frameworks` | Comparativo entre Angular, Vue e React |
| Usuários/API | `/usuarios` | Exemplo prático de consumo de API |
