🧪 Testes com Karma e Jasmine

Karma

É o executor de testes usado pelo Angular. Ele abre um navegador e executa os testes criados no projeto.

Jasmine

É o framework de testes que fornece comandos como describe(), it() e expect().

Comando

Para rodar os testes da aplicação, use: 

ng test
